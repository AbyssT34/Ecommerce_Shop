## 3.3 Luồng Phụ

### Luồng A: Duyệt Sản phẩm → Xem Món có thể nấu

```
Trang Danh mục  →  Trang Sản phẩm  →  "Món có thể nấu từ sản phẩm này"
                                              │
                                              ▼
                                       Trang Công thức
```

### Luồng B: Tìm kiếm trực tiếp

```
Thanh Tìm kiếm  →  Kết quả (Sản phẩm + Món ăn)  →  Công thức/Sản phẩm
```

### Luồng C: Lịch sử & Yêu thích

```
Tài khoản  →  Món đã xem/yêu thích  →  Công thức  →  Mua lại
```
